### 2、AppDelegate

`AppDelegate.m`
//这个方法返回 热更新开关的域名 不同厅主的开关域名不一样
```objective-c
+(NSArray *)getBBQArray {
#pragma mark ⚽︎ ❤️❤️❤️ ⚽︎ 一定要返回一组正确请求地址
  return @[@"https://www.c-3.cc",
           @"https://www.c-6.cc",
           @"https://www.c-8.cc"];
}
```

我们会以该接口返回的bbq字段 含有 @"SueL" 字符作为已开启APP的标志 请求参数包括：appId、版本号、设备类型
如需修改 可以在 AppDelegate+JDBase.m 中修改
